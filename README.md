## Linguagem utilizada
### Variáveis
```
  Utilize conts ou let para declaração de variáveis
```
```
//Ruim
var nome = 'Samara';

//Bom
let nome = 'Samara';
```

---
### Definição de funções 
```
  Evite o uso de else
```
```
  //Ruim 
  if(a ==b){
    console.log('Iguais');
  } else{
    console.log('Diferentes');
  }

  //Bom
  if (a == b){
    console.log('Iguais');
  }

  if(a != b){
    console.log('Diferentes');
  }
```
